## Update a Recurring Request

<ApiEndpoint method="PATCH" path="/v2/request/{requestId}" />

## Path Parameters

<ParamField path="requestId" type="string" required>
  The requestId for the recurring request
  
  Example: `01e273ecc29d4b526df3a0f1f05ffc59372af8752c2b678096e49ac270416a7cdb`
</ParamField>

## Headers parameters

<ParamField header="x-api-key" type="string" required>
  API key for authentication
</ParamField>


## Response

Returns `200 OK` on successful update.

## Error Responses

<ResponseField name="401" type="object">
  Unauthorized - invalid API key
</ResponseField>

<ResponseField name="404" type="object">
  Request not found
</ResponseField>

<ResponseField name="429" type="object">
  Too Many Requests - rate limit exceeded
</ResponseField>

<ResponseExample>

```

```

</ResponseExample>
